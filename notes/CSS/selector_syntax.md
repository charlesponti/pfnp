
```css
p {
  height: 200px;
}
```

The above CSS would make all ` <p> ` elements 200px tall.

## Child Selector Syntax

```html
<ul>
  <li>Bears</li>
  <li>Cats</li>
  <li>Gophers</li>
</ul>
```


```css
  ul li {
    height: 200px;
  }
```

The above CSS would make ` <li> ` elements 200px tall, only if they are
inside of a ` <ul> `.
